Guia Passo a Passo
1. Preparar seu Grupo no Telegram
  • Crie um grupo no Telegram (se ainda não tiver)
  • Configure o grupo como privado
  • Prepare uma mensagem de boas-vindas
2. Adicionar o Bot
  • Procure por @televipbra_bot no Telegram
  • Adicione o bot como administrador do grupo
  • Dê permissões para gerenciar usuários e mensagens
3. Obter o ID do Grupo
  • No grupo, digite o comando: /setup
  • O bot responderá com o ID do grupo
  • Copie e cole o ID no formulário acima
4. Configurar Planos e Preços
  • Defina diferentes planos (mensal, trimestral, etc.)
  • Configure preços competitivos
  • Você pode adicionar ou modificar planos depois
Guia Passo a Passo
1. Criar Canal no Telegram
  • Crie um canal no Telegram (privado)
  • Configure o canal com nome e descrição
2. Adicionar o Bot como Administrador
  • Procure por @televipbra_bot no Telegram
  • Adicione o bot como administrador do canal
  • Dê permissões para gerenciar usuários
3. Obter o ID do Canal
  • Adicione @userinfobot ao canal
  • Ele mostrará o ID do canal (ex: -1001234567890)
  • Copie e cole o ID no formulário acima
4. Configurar Planos e Preços
  • Defina diferentes planos (mensal, trimestral, etc.)
  • Configure preços competitivos
